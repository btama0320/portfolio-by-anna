販売管理システム

Laravel × MySQLで構築した業務効率化システム

概要

市場でのアナログ業務を効率化するために開発した販売管理システムです。
売掛伝票の入力・集計・ユーザー管理・ログイン機能など、現場運用を意識した設計を重視しています。

主な機能

  • ログイン認証（社員ID＋初期パスワード）
  • 売掛伝票入力（行番号・スクロール・集計欄付き）
  • ユーザー管理（管理者のみアクセス可能）
  • パスワード変更・初回変更フラグ
  • 操作ログ・レポート管理

技術構成

フレームワークLaravel 10
データベースMySQL（Railway連携）
デプロイRender
開発環境VS Code / GitHub / Windows
